System Requirements

Your server need to meet the following requirements in order to run RentArea

  • PHP >= 5.6.4
  • MYSQL 5+
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

Optional Requirements:

  • Composer
  • NPM / YARN

Migrate files using any FTP program (ie. FileZilla) or the webinterface of your webhost

Migrate folders and files from zip file(RentArea-project.zip) with any FTP program (ie. FileZilla) or the webinterface of your webhost.

  1. Connect to your web server using any FTP program (ie. FileZilla) or the webinterface of your webhost and upload all files and folders
    which are inside the "rentarea" folder from the .zip file you have downloaded from Github and put in the public_html folder.
     

  2. The folder "public_html" and it's subfolders must have write permissions, so just give the folder write permissions (777).
     

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Migrate project files via Cpanel

Migrate folders and files from zip file(RentArea-project.zip) with Cpanel.

  1. Access your Cpanel account and login in with your credentials. Then go to File Manager.
     

  2. Go to public_html folder.
     

  3. Upload " RentArea-project" the .zip file you have downloaded from Github.
     

  4. Extract "RentArea-project" .zip file you have downloaded from Github.
     

  5. Your public_html folder should have all these files and folders.
     

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Creating the Database for the application from Cpanel

Create/Configure/Import the database

I. Create Database

  • Create the database using MySQL® Database Wizard
     

II. Import Database

  • Import the database 'rentarea_db.sql' from the package you have downloaded.
     

III. Configure the .env file

  1. Make sure that we have the 'show hidden files' checked
     

  2. Edit the .env file and paste the correct credentials
